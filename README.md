Customer churn is one of the most critical challenges in the telecom industry. Acquiring new customers is significantly more expensive than retaining existing ones. Therefore, understanding customer behavior and identifying churn patterns is essential for improving retention strategies.
This project presents an interactive Power BI dashboard that analyzes telecom customer churn data to uncover key insights, high-risk customer segments, and major churn drivers.
The goal of this project is to transform raw telecom customer data into meaningful, data-driven insights using Business Intelligence tools.
